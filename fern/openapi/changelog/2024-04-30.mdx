subtitle: Adding Support for NFT Checkout on Apex Protocol
---

**Starting today, April 30, 2024, Crossmint supports NFT Checkout on Apex Protocol.**

## Importing an Apex Protocol smart contract

- Go to the “Collections” tab on Crossmint’s Console.
- Create a new collection.
- Select "Import an existing contract" and select the chain as "Apex".
- Now enter the contract address and then create the collection.
- After this, you will be able to enable NFT Checkout for your contract.

If you have further questions regarding this, contact [Crossmint Support]().
